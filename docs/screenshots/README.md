# Release screenshots

Add PNG screenshots here before tagging a release. They are attached to GitHub releases when present.

| File | What to capture |
|------|-----------------|
| `form.png` | Public withdrawal form on the front end (with sample fields visible) |
| `admin-requests.png` | Admin **Withdrawal requests** list with a few sample rows |
| `settings.png` | Plugin **Settings** page (any tab that shows key options) |

Tips:

- Use a staging site with test data, not real customer information.
- Recommended width: 1200–1600 px.
- PNG format only (`*.png`).
